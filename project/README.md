# 💼 Personal Resume (HTML Only)

## 📌 Overview
This project is a personal resume created using only HTML with inline CSS styling.  
It presents my education, experience, skills, projects, and certifications in a structured format.

---

## 🚀 Features
- Single-file project (no external CSS or JS)
- Clean and simple layout
- Easy to edit and customize
- Lightweight and fast

---

## 🛠️ Technologies Used
- HTML5
- Inline CSS
